# Daily Work Todo

## Run locally

Open `index.html` directly, or use any static server.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep the default static-site settings.
4. After deployment, open the generated URL.

## Notes

- Tasks and archive data are stored in the current browser's `localStorage`.
- "Export Markdown" downloads the current archive as a `.md` file.
- Word import uses browser-loaded libraries and requires network access to jsDelivr.
